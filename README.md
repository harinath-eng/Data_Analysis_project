# Data_Analysis_project
# Company Layoffs Data Analysis Project

## Overview
This project analyzes global tech layoffs from 2020 to 2023 using a Kaggle dataset.  
The goal is to identify trends in layoffs across industries, countries, company stages, and time periods using SQL and data visualization tools.

---

## Dataset Source
- Kaggle Layoffs Dataset: https://www.kaggle.com/datasets/swaptr/layoffs-2022

---

## Tools Used
- SQL (Data Cleaning & Analysis)
- MySQL (Database Management)
- Tableau / Power BI (Visualization)
- Excel (Basic data inspection)

---

## Data Cleaning Process
The dataset was cleaned before analysis:
- Removed duplicate records
- Handled NULL values
- Standardized inconsistent data formats
- Filtered irrelevant or incomplete entries

---

## Key Analysis Questions
- Which companies had the highest layoffs?
- Which industries were most affected?
- How did layoffs change over time (2020–2023)?
- Which countries experienced the most layoffs?
- Which company stages (Startup, IPO, etc.) were most impacted?

---

## Key Insights
- Layoffs increased significantly during economic downturn periods.
- Tech industry was the most affected sector.
- Certain funding stages showed higher vulnerability to layoffs.
- USA had the highest number of affected companies.

---

## Visualizations
Dashboards were created using Tableau / Power BI to show:
- Year-wise layoff trends
- Industry-wise impact
- Top affected companies
- Geographic distribution of layoffs

---

## Limitations
- Dataset covers only 2020–2023, so long-term trends cannot be fully analyzed.
- Some assumptions were made where company status data was incomplete.
